# Exercice SWOT

## Enoncé

Vous travaillez au service marketing d'une société de cours particuliers.

Cette société propose à ses clients des cours personnalisables essentiellement en visio, avec des points de coordination mensuels avec les parents/tuteurs pour garantir un meilleur suivi des étudiants.

**Consigne :** Réalisez un diagnostic interne et externe puis concluez avec un SWOT sur ce projet entrepreneurial.
